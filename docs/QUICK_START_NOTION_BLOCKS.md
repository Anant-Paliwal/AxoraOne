# Quick Start: Notion-Style Blocks

## 🚀 Get Started in 5 Minutes

### Step 1: Create a New Page
1. Navigate to Pages
2. Click "New Page"
3. Add title and icon

### Step 2: Add Your First Block
1. Click "Add Block" button
2. Select "Database" from picker
3. Block appears instantly!

### Step 3: Reorder Blocks
1. Hover over any block
2. Drag handle appears on left (⋮⋮)
3. Drag up or down to reorder
4. Auto-saves automatically

### Step 4: Add an Image
1. Click "Add Block"
2. Select "Image"
3. Paste image URL
4. Image appears at 100% width

### Step 5: Resize the Image
1. Hover over image
2. Drag left or right edge
3. Or click preset: 25%, 50%, 75%, 100%
4. Done!

## 🎯 Common Tasks

### Add Multiple Blocks
```
1. Click "Add Block"
2. Select block type
3. Repeat for each block
4. Drag to reorder
```

### Create Database
```
1. Add "Database" block
2. Click "Add Column" to add fields
3. Click "Add Row" to add data
4. Click column headers to sort
```

### Link Pages
```
1. Open page sidebar
2. Click "Add Link"
3. Search for page
4. Select link type
5. Create link
```

### Resize Media
```
Method 1: Drag edges
- Hover over image
- Drag left/right edge

Method 2: Presets
- Click 25%, 50%, 75%, or 100%
```

## 💡 Pro Tips

### Tip 1: Block-First Layout
- Add blocks at top for structured data
- Add text below for explanations
- Text editor hides when not needed

### Tip 2: Fullscreen Preview
- Click any image to view fullscreen
- Press ESC or click outside to close

### Tip 3: Quick Sizing
- Use presets for consistent widths
- 50% for side-by-side images
- 100% for hero images

### Tip 4: Organize with Blocks
- Database for tables
- Calendar for events
- Gallery for images
- Timeline for history

## 🐛 Troubleshooting

### Block not dragging?
- Make sure you're hovering to see drag handle
- Click and hold the handle (⋮⋮)
- Drag up or down

### Image not resizing?
- Hover over image to see resize handles
- Drag the left or right edge
- Or use preset buttons below image

### Text editor not showing?
- It's hidden when you only have blocks
- Start typing to make it appear
- Or add text content to show it

### Auto-save not working?
- Auto-save triggers after 30 seconds
- Or when you manually click "Save"
- Check for "Saved" indicator in header
