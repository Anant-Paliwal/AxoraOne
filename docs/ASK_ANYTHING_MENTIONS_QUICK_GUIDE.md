# @ Mentions Quick Guide

## What Are @ Mentions?

@ mentions let you reference specific items **from your current workspace** in Ask Anything queries for more accurate, contextual AI responses.

## 🔒 Workspace Isolation

**Important**: You can ONLY mention items from your **current workspace**. No cross-workspace access is allowed for security and clarity.

## How to Use

### Basic Usage
1. Type `@` in the Ask Anything input
2. A dropdown appears showing items from your **current workspace only**
3. Click an item or keep typing to filter
4. The mention is inserted into your query

### Type Filters (Optional)
- `@page:` - Show only pages from current workspace
- `@task:` - Show only tasks from current workspace
- `@skill:` - Show only skills from current workspace

Short versions: `@p:`, `@t:`, `@s:`

## Example Queries

### Learning
```
"Summarize @SQL Basics"
"Create a quiz from @Python Functions"
"Explain @Recursion in simple terms"
"Generate flashcards for @Data Structures"
```

### Tasks
```
"What's the status of @Deploy API?"
"Create subtasks for @Complete Migration"
"Show tasks related to @JavaScript skill"
```

### Comparison
```
"Compare @React and @Vue"
"What's the difference between @Machine Learning and @Deep Learning?"
"How do @Authentication and @Authorization relate?"
```

### Content Creation (BUILD mode)
```
"Create a page combining @Topic A and @Topic B"
"Build practice tasks for @Python skill"
"Generate a learning path from @Beginner to @Advanced"
```

## Tips

✅ **Current Workspace Only**: Only items from your active workspace appear
✅ **Multiple Mentions**: You can mention multiple items in one query
✅ **Any Mode**: Works in Ask, Explain, Plan, and Build modes
✅ **Smart Context**: AI gets full content of mentioned items
✅ **Secure**: Cannot access items from other workspaces

## Benefits

🎯 **Precise**: No ambiguity about which item you mean
🚀 **Fast**: Autocomplete makes it quick to find items
🧠 **Smart**: AI understands full context of mentioned items
🔒 **Secure**: Strict workspace isolation protects your data
🔗 **Connected**: Easily relate different workspace items

## Workspace Indicator

The dropdown shows which workspace you're in:
```
@ Mention in 📊 Data Science
```

This reminds you that only items from "Data Science" workspace will appear.

## Empty State

If no items are found:
```
No items found in [Your Workspace]
Try: @page:, @task:, or @skill:
```

## Keyboard Shortcuts

- `@` - Open mentions dropdown
- `Type` - Filter items
- `Click` - Select item
- `Esc` - Close dropdown

---

**Pro Tip**: Start your query with a mention to make it super clear what you're asking about!

**Security Note**: You can only mention items from your current workspace. To work with items from another workspace, switch to that workspace first.
