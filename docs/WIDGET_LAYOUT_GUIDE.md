# Widget Layout Options - Visual Guide

## 🎯 Quick Access

The **View** button is always visible at the top of your homepage - no need to enter edit mode!

```
┌─────────────────────────────────────┐
│  [View ▼]  [Customize]              │
│                                     │
│  Your Widgets Below...              │
└─────────────────────────────────────┘
```

## 📱 Single Column Layout

**Best for:** Focus mode, mobile devices, presentations

```
┌─────────────────────────────────────┐
│                                     │
│         Widget 1                    │
│                                     │
├─────────────────────────────────────┤
│                                     │
│         Widget 2                    │
│                                     │
├─────────────────────────────────────┤
│                                     │
│         Widget 3                    │
│                                     │
└─────────────────────────────────────┘
```

**Features:**
- One widget per row
- Maximum focus on each widget
- Great for mobile or when you want to see full details
- Vertical scrolling

## ⚏ 2x2 Grid Layout

**Best for:** Balanced view, medium screens, dual monitors

```
┌──────────────────┬──────────────────┐
│                  │                  │
│    Widget 1      │    Widget 2      │
│                  │                  │
├──────────────────┼──────────────────┤
│                  │                  │
│    Widget 3      │    Widget 4      │
│                  │                  │
├──────────────────┼──────────────────┤
│                  │                  │
│    Widget 5      │    Widget 6      │
│                  │                  │
└──────────────────┴──────────────────┘
```

**Features:**
- Two columns on tablet and desktop
- Balanced information density
- Good compromise between focus and overview
- Works well on most screen sizes

## ⊞ 3x3 Grid Layout (Default)

**Best for:** Maximum information density, large screens, dashboards

```
┌────────────┬────────────┬────────────┐
│            │            │            │
│  Widget 1  │  Widget 2  │  Widget 3  │
│            │            │            │
├────────────┼────────────┼────────────┤
│            │            │            │
│  Widget 4  │  Widget 5  │  Widget 6  │
│            │            │            │
├────────────┼────────────┼────────────┤
│            │            │            │
│  Widget 7  │  Widget 8  │  Widget 9  │
│            │            │            │
└────────────┴────────────┴────────────┘
```

**Features:**
- Three columns on desktop
- Maximum information at a glance
- Perfect for large monitors
- Dashboard-style overview

## 🎨 Spacing Options

Combine any layout with spacing preferences:

### None (Notion-style)
```
[Widget][Widget][Widget]
[Widget][Widget][Widget]
```
- No gaps between widgets
- Clean, seamless flow
- Maximum screen usage

### Compact
```
[Widget] [Widget] [Widget]
[Widget] [Widget] [Widget]
```
- Small 8px gaps
- Subtle separation
- Still space-efficient

### Comfortable
```
[Widget]  [Widget]  [Widget]

[Widget]  [Widget]  [Widget]
```
- Larger 16px gaps
- More breathing room
- Easier to distinguish widgets

## 🔄 Responsive Behavior

### Single Column
- **All devices:** 1 column

### 2x2 Grid
- **Mobile:** 1 column
- **Tablet+:** 2 columns

### 3x3 Grid
- **Mobile:** 1 column
- **Tablet:** 2 columns
- **Desktop:** 3 columns

## 🎯 Quick Tips

1. **Switch layouts anytime** - Click View button (always visible)
2. **No need to save** - Changes apply instantly
3. **Works with all widgets** - Any widget adapts to your layout
4. **Responsive** - Automatically adjusts on smaller screens
5. **Combine with spacing** - Mix layout + spacing for perfect look
