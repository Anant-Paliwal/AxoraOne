## 🚀 Quick Start Guide

### Change Workspace Color:
1. Open Settings
2. Go to Preferences tab
3. See 18 colors in grid
4. Click any color
5. Watch workspace change instantly
6. Click "Save Preferences"

### Adjust Font Size:
1. Open Settings
2. Go to Preferences tab
3. See 3 size options with "Aa" preview
4. Click preferred size
5. See live preview below
6. Click "Save Preferences"

### Manage Notifications:
1. Open Settings
2. Go to Notifications tab
3. See 6 notification types
4. Toggle switches on/off
5. Click "Save Notification Settings"

### Export Page as PDF:
1. Open any page
2. Click ⋯ (more options menu)
3. Click "Export as PDF"
4. Print dialog opens
5. Choose "Save as PDF"
6. Select location and save

### Export Page as Markdown:
1. Open any page
2. Click ⋯ (more options menu)
3. Click "Export as Markdown"
4. File downloads automatically
5. Open in any markdown editor
