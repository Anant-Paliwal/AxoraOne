## 🔄 Installation Flow

```
1. Browse Marketplace
   ↓
2. Find Skill (search/filter/browse)
   ↓
3. Click "Add Skill" or view details
   ↓
4. Skill installed to workspace
   ↓
5. Appears in main Skills list
   ↓
6. Ready to use immediately
```

---

## 💡 Recommended Skills Logic

```
IF user has skills in category "startup"
  → Recommend more "startup" skills
  → Reason: "Based on your startup skills"

IF user has skills in category "execution"
  → Recommend more "execution" skills
  → Reason: "Based on your execution skills"

IF user has NO skills
  → Recommend top 3 most popular
  → Reason: "Popular choice"
```

## 🎯 Use Cases

### 1. New User Onboarding
```
User: "I want to start tracking skills"
Action: Open marketplace → Browse "Popular" → Install 2-3 skills
Result: Instant skill library without manual creation
```

### 2. Category Expansion
```
User: Has planning skills, wants execution skills
Action: Filter by "Execution" → Install Focus Protector
Result: Balanced skill portfolio
```

### 3. Bundle Installation
```
User: "I'm starting a startup"
Action: Install "Startup Pack" bundle
Result: 3 related skills (Idea Validator, MVP Clarity, Experiment Planner)
```

### 4. Skill Discovery
```
User: "What skills are available?"
Action: Browse marketplace, read descriptions
Result: Learn about skill types and capabilities
```

## 🚀 Quick Actions

| Action | Shortcut |
|--------|----------|
| Open Marketplace | Click "Marketplace" button |
| Search Skills | Type in search bar |
| Filter Category | Click category in sidebar |
| View Details | Click chevron (>) on card |
| Install Skill | Click "+ Add Skill" |
| Close Marketplace | Click X or press Escape |

## ✨ Visual Hierarchy

```
1. Header (Title + Search + Sort)
   ↓
2. Recommended Section (if applicable)
   ↓
3. All Skills Grid (filtered/sorted)
   ↓
4. Sidebar (Categories, Top Rated, Bundles)
```

---

## 🎨 Design Elements

- **Cards**: Rounded corners, hover effects
- **Icons**: Large emoji icons for visual recognition
- **Colors**: Primary for actions, muted for metadata
- **Spacing**: Generous padding for readability
- **Typography**: Clear hierarchy (title > description > metadata)
- **Feedback**: Loading states, success toasts

---

## 📱 Responsive Design

```
Desktop (lg):  3 columns grid
Tablet (md):   2 columns grid
Mobile (sm):   1 column grid

Sidebar:       Collapsible on mobile
Search:        Full width on mobile
Cards:         Stack vertically on mobile
```

## 🎯 Success Indicators

After installing a skill:
- ✅ Toast notification: "Skill installed successfully!"
- ✅ Skill appears in main Skills page
- ✅ Skill has all properties (category, level, purpose, etc.)
- ✅ Skill is ready to use immediately
- ✅ Marketplace closes automatically
